## SETUP & RUN PROJECT

- Clone repository: 
```
git clone https://github.com/ThaoLyyy/HTML-CSS-Training.git
```
- Checkout branch:
``` 
git Checkout develop
```
- Open Folder:
```
cd practice-two
```
- Install packages dependencies :
```
npm install
```
- Run Webpage:
```
npm start
```
- Open in browser:
```
~http://localhost:1234~
```
